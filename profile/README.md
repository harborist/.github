# Harborist

Welcome to the Harborist organization. We are dedicated to maintaining the security and integrity of open source software through responsible vulnerability disclosure and comprehensive security advisories.

[![CNA Scorecard](https://cnascorecard.org/badges/harborist-both.svg)](https://cnascorecard.org/cna/cna-detail.html?shortName=harborist)

## Security Advisories

For detailed security advisories and vulnerability information, please visit our [advisories repository](https://github.com/harborist/advisories).

## About

Harborist operates as a CVE Numbering Authority (CNA), working to identify, track, and disclose security vulnerabilities in open source software projects. Our mission is to enhance the security posture of the open source ecosystem through transparency and collaboration.
